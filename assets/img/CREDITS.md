# Background image provenance

Per-section parallax backgrounds. Personal photos are used where they exist;
the rest are free-license stock, processed (resized + recompressed) for the web.

To swap any background, drop a replacement JPG at the same path (or change the
`background-image` URL in the matching `.panel--*` rule in `assets/css/main.css`).
Prefer a personal photo over stock wherever one is available.
